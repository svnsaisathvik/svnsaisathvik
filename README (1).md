## 🚀 Featured Projects

<br/>

### 🛰️ Cloud-Free Satellite Imagery Generation
> *Deep Learning · Earth Observation · Computer Vision*

[![Repo](https://img.shields.io/badge/View%20Repo-Cloud--Removal-7C3AED?style=for-the-badge&logo=github&logoColor=white)](https://github.com/svnsaisathvik/Cloud-Removal)
&nbsp;
![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![PyTorch](https://img.shields.io/badge/PyTorch-EE4C2C?style=flat-square&logo=pytorch&logoColor=white)

A deep learning pipeline that reconstructs **cloud-free optical satellite images** from multi-temporal time series. Built a custom **TimeGate** attention mechanism to learn which pixels are cloud-covered, followed by two reconstruction paths — a **residual U-Net** and a fine-tuned **TerraMind foundation model** (IBM × ESA). Achieved **SSIM > 0.90** on validation data.

[![Repo](https://img.shields.io/badge/View%20Repo-GSC-059669?style=for-the-badge&logo=github&logoColor=white)](https://github.com/svnsaisathvik/GSC)

[![Challenge 1B](https://img.shields.io/badge/View%20Repo-Challenge%201B-EA580C?style=for-the-badge&logo=github&logoColor=white)](https://github.com/svnsaisathvik/Adobe_India_Hackathon_Challenge_1B)

### 🎙️ ML Fiesta — Kannada ASR & Question Answering
> *NLP · Speech Recognition · Regional Language AI*

[![Repo](https://img.shields.io/badge/View%20Repo-ML--Fiesta-7C3AED?style=for-the-badge&logo=github&logoColor=white)](https://github.com/svnsaisathvik/ML-Fiesta)
&nbsp;
![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![OpenAI](https://img.shields.io/badge/Whisper-412991?style=flat-square&logo=openai&logoColor=white)

Built at **Synergy24** — an ASR + QA pipeline to preserve indigenous knowledge on sandalwood cultivation in Karnataka. Fine-tuned **OpenAI Whisper** on colloquial Kannada YouTube audio, then built a sentence-level retrieval system so users can query the entire audio corpus in their native language and get targeted answers back.
